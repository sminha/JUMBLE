# Jumble <img src="https://github.com/sminha/JUMBLE/blob/develop/client/public/favicon.png" width="110" align="left" />

의류 사입내역 관리를 보다 편리하게

<br/>

<div align="center">

        유료 솔루션 사용에 부담감을 느끼는 초기 쇼핑몰 사장님들은 어쩔 수 없이 사입내역 관리 수단으로 엑셀을 택하게 되죠.
        그러나 이 방식은 금방 한계에 부딪히게 됩니다. Jumble은 이러한 관리의 사각지대를 해소하기 위해 탄생했습니다.
        
        🧾 사입내역 관리: 직관적인 UI로 방대한 내역들을 보다 편리하게 관리할 수 있습니다.
        🫆 미송건 관리: 불규칙한 미송 입고 상황을 간편하게 업데이트하고 추적할 수 있습니다.
        📊 대시보드 기능: 상품별, 거래처별 통계 등 다양한 통계를 한 눈에 확인할 수 있습니다.
        
        Jumble과 함께 스마트한 사입내역 관리를 시작해 보세요!

</div>

<br />

## *️⃣ 핵심 기능

<h3>MLLM 기반 자동 입력</h3>

- 영수증 이미지 업로드 시 VLM이 텍스트를 추출하고 분석하여 입력 폼을 자동으로 채워줍니다.
- 수기 입력 대비 입력 시간 60% 단축 (10초 → 4초)
